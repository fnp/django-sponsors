Dependencies
============

 * Django
 * django-jsonfield
 * PIL


Usage
=====

1. Add to INSTALLED_APPS and migrate.
2. Define your sponsor pages in the admin.
3. Use the {% sponsor_page %} template tag from sponsor_tags.
